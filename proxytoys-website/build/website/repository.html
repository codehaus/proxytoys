Anonymous Subversion Access

This ProxyToys source can be checkout out anoymously from this Subversion repository:

svn://svn.proxytoys.codehaus.org/proxytoys/scm/trunk/proxytoys

Example

svn co svn://svn.proxytoys.codehaus.org/proxytoys/scm/trunk/proxytoys

Developer Subversion Access

For commit access, developers require an SSH account.

svn+ssh://svn.proxytoys.codehaus.org/scm/proxytoys/trunk/proxytoys
svn+ssh://svn.proxytoys.codehaus.org/scm/proxytoys/trunk/proxytoys-website
