https://svn.codehaus.org/proxytoys/trunk/proxytoys

Anonymous Subversion Access

The ProxyToys source can be checkout out anonymously from the Subversion repository:

svn co svn://svn.proxytoys.codehaus.org/proxytoys/scm/trunk/proxytoys

Developer Subversion Access

For commit access, developers require an SSH account. Checkout with:

svn co svn+ssh://svn.proxytoys.codehaus.org/scm/proxytoys/trunk/proxytoys
